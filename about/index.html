About

As a Senior Data Scientist, based in Tampa, FL, I specialize in developing production-level python code, utilizing the apache-spark framework for application development, custom built machine learning algorithms and apache-airflow to automate workloads, while ensuring all is operational and functional on AWS EMR and AWS EC2. The following sections covered my secular and academic pursuits.

Secular Pursuits

I currently work for Paylocity as a Data Scientist.

Previously, I worked for Nielsen as a Senior Data science, where I was managing code repositories, which are 100% responsible for the application of Fusion methodology, and heavily involved with the development, and transition from SAS, of the Total Media Fusion into Python and PySpark. This particular fusion is responsible for taking data regarding various media platforms and combining these data sources, with the sole purpose of representing the entire media universe. Majority of the time, I focus on developing code utilizing modularity, portability, scalability, just-in-time compilation, and parallelism, in order to help deliver amazing products to various clients.

I previously worked as an Adjunct Instructor for Stony Brook University and Suffolk County Community College. During that time, I taught Single-Variable Calculus, Differentiable Calculus and Algebra 1.
